Isoloated scope in directive

    
    In the page we will be discussing about the directive scope only...
    scope can be like:-
        1. scope:true
        2. scope:false
        3. scope:{}
        
 Example:-

        return {
            restrict:'EA',
            scope: true,'
            template: " Name: {{name}} Address: {{address}}"
                };
        here in this example i have used scope :true
        
scope:true  means

        if you use true it looks from the directive scope itself not from parents. But, any changes maid in paren
        with the same property will reflect to the directive(child) also.
        
scope:false  means

        if you use false it looks from the directive scope as well as parent.
        if property changes in child or in parents it reflects in both scope (parent and child).
        Means:- if you change in directive(child) it changes into parents and vice versa.
        
scope:{}  means

        It does not look for parents properties it looks for only it's directive scope.
